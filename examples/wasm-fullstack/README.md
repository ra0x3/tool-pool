# WASM Fullstack Examples

The WASM fullstack examples have been split into separate directories:

- **[wasm-fullstack-v1](../wasm-fullstack-v1/)** - Simple in-memory implementation without external dependencies
- **[wasm-fullstack-v2](../wasm-fullstack-v2/)** - Advanced implementation with real PostgreSQL and HTTP support (requires WasmEdge runtime)

Please see the respective directories for build and run instructions.
